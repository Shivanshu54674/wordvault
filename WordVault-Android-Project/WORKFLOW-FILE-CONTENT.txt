If the hidden ".github" folder did not upload to GitHub (it is hidden on
most computers), create it manually:

On your GitHub repository page:
1. Click "Add file" -> "Create new file"
2. In the filename box type EXACTLY:  .github/workflows/build.yml
   (typing the / automatically creates the folders)
3. Paste everything below the line into the big text box
4. Click "Commit changes"

----------------------------------------------------------------------
name: Build WordVault APK

on:
  push:
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout code
        uses: actions/checkout@v4

      - name: Set up Java 17
        uses: actions/setup-java@v4
        with:
          distribution: temurin
          java-version: '17'

      - name: Set up Gradle
        uses: gradle/actions/setup-gradle@v3
        with:
          gradle-version: '8.7'

      - name: Build APK
        run: gradle assembleDebug

      - name: Upload APK
        uses: actions/upload-artifact@v4
        with:
          name: WordVault-APK
          path: app/build/outputs/apk/debug/app-debug.apk
